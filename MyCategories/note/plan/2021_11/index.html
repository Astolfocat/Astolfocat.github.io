# 2021.11.26 Fri

从 24 到 26 号三天，吃了一顿早饭，一盒泡芙，二分之一的蛋糕，一个好丽友派，居然没有感觉到饿，好神奇（虽然昨晚全身乏力，但是今天起来好多了）而且晚上天文知识竞赛结束还有庆生蛋糕吃好耶，群友还给我送来了好多零食，好感谢，草莓汉堡包派超好吃的，还有酸奶泡芙！！！遗憾的是今天又停电一天（悲）。下午四点就去准备天文知识竞赛，还要穿正装，可恶，正装一点也不适合我（啊不对，我不好看穿什么都不好看，不该怪正装的）。然后裤子腰巨大没有腰带，机智如我拿数据线作腰带 233333。晚上计分出了一点小问题，还是缺人手啊。
今年的天文知识竞赛没有去年办的好了，说实话有点悲哀，大家都在努力，可惜还是没做好。为什么呢？去年节奏有很多地方没做好，赛制也发生了很大变化，今年在原有的基础上不但没有做好，反而丢失了去年设计的初衷。策划和演示几乎借用去年的模板，但是却做得更差了。天协是一个大家一起玩天文的地方，不是给学校攒影响力的地方。做活动，应付一下，结束，周而复始，天协已经变成一个死胎了。没有主心骨，没有带头的人。很多问题学长都提了，但是并没有引起重视。不热爱的人占的太多了。15 级的人仍在自豪，18 级的还在回顾，20 级的已经大部分开始划水了。三年后不会有人在为了自己发展了一个社团而骄傲，而是在拿着任职证明说自己有过组织管理的能力。天协缺少真正愿意花精力去把活动做好，把天协做好的人。很多学长留给后面的人的文件都浪费了，组织部方面，决策这种事情专门有过总结，各种问题给出过检验后的改进方法，但并没有人在意。之前的资料放云盘全员学习的计划也没有执行。天协的图书馆计划（学长带着秘书处做了很久来着）也没有按预期效果执行的样子。部门做了培训，但约等于没做。经验文件留了下来，但也约等于没留。形式类的东西不应该占用太多时间，不应该本末倒置。反思会议材料应该准备，大活动后应该反思这次活动哪里做的不好哪里存在问题，留出对策写进总章。
可恶，被杰哥狠狠地伤害了。阿福还是败了 ((((*｡_｡)_

# 2021.11.27 Sat

狠狠地睡死了，下午两点多才爬起来，然后就忙着去归还抢答器然后去参加天协高层团建了。今晚吃得真香，后面几天又可以不吃了（好耶），大家玩狼人杀也好有趣～印象最深刻的是有一局还剩八个人，四个神职三个平民一个狼，然后神职把猎人票出去了，猎人带走了守卫，女巫毒死了白痴。最后差点让狼人翻盘 233333。好好休息准备明天的比赛！

# 2021.11.28 Sun

五个小时的程序设计竞赛，题目确实很有意思，但是我的基础明显还不够牢固（尤其是第一题概率论的题居然没看出来）。下午去虚零家恰火锅！！！好耶。自己处理食材，自己准备原材料…… 第一次在室内自己做火锅呢（可惜没有辣碟）。他们玩真心话大冒险，我帮他们下菜挑菜（怎么看都是我赢了，保住了自己的安全（雾）），杰哥调酒好好喝，不喝酒的阿福都喝了一杯。虚零杰哥 Miakiu 鲨鲨赵谢，好耶～

几乎满课，数据结构要准备写排序作业了，概率论也开始进入了看不懂的三大分布了。模电继续摆烂，回宿舍把选修课论文水了出来。然后继续摸鱼。
